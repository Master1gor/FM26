Football Manager rule files for OFC v26.1

In this package, you will find rule files for OFC Member Nations.

I did my best to use realistic formats for all the leagues that are included. The rule files contain:

  • real divisions and cups with real formats from 2025/26
  • teams from the 2025/26 season
  • all nations have transfer windows, registration periods, tv dates, match rules, prize money - when I was able to find this in my online research

Compatibility

  • FM Database 26.2.0 (only)

Installation Guide

Copy the .fmf files for the nations you want into the folder "<My Documents>\Sports Interactive\Football Manager 2026\editor data\". The folder can be different if you have changed it in the FM26 preferences. The leagues should then appear the next time you start a new game in FM26 and chose advanced setup.
Make sure you have the checkbox "add players to playable teams" checked. This is recommended, otherwise you may end up with a lot of greyed-out players.

  • (!) Please use only 1 file for OFC Champions league.
  • OFC.fmf : OFC Champions League implemented in 2023 format using coefficients. AFter 3 years (from 2026 onwards), the ranking based on coefficients will be used so that there are no big shifts right after the game starts.
  • OFC_ALL.fmf : OFC Champions League implemented in 2023 format using coefficients and including teams from Tuvalu and Kiribati. AFter 3 years (from 2026 onwards), the ranking based on coefficients will be used so that there are no big shifts right after the game starts.
  • OFC_NoCoefficients.fmf : OFC Champions League implemented in 2023 format. No coefficients are used, the nation ranking is static.
  • OFC_ALL_NoCoefficients.fmf : OFC Champions League implemented in 2023 format including teams from Tuvalu and Kiribati. No Coefficients are used, the nation ranking is static.
