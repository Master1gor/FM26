Football Manager rule files for CONMEBOL v26.1

In this package, you will find rule files for CONMEBOL Member Nations.

I did my best to use realistic formats for all the leagues that are included. The rule files contain:

  • real divisions and cups with real formats from 2025/26
  • teams from the 2025/26 season
  • all nations have transfer windows, registration periods, tv dates, match rules, prize money - when I was able to find this in my online research

Compatibility

  • FM Database 26.0.0 (only)

Installation Guide

Copy the .fmf files for the nations you want into the folder "<My Documents>\Sports Interactive\Football Manager 2026\editor data\". The folder can be different if you have changed it in the FM26 preferences. The leagues should then appear the next time you start a new game in FM26 and chose advanced setup.
Make sure you have the checkbox "add players to playable teams" checked. This is recommended, otherwise you may end up with a lot of greyed-out players.

CONCACAF Continental Competitions v26.1
[Cup] CONCACAF Champions Cup (*REPLACED)
[Cup] Leagues Cup
[Cup] Canadian Championship (*REPLACED)
[Cup] CONCACAF Central American Cup (*REPLACED)
[Cup] CONCACAF Caribbean Cup (*REPLACED)
[Cup] Caribbean Club Shield (*ADDED)

  • I have recreated the CONCACAF Club Competitions because the in-game versions had some issues. Here is a list of what my version (should) fix:
    • Finalists from Caribbean Club Shield qualify for Caribbean Cup
    • Central American Cup no longer has a host nation and this fixes schedule issues with central american leagues (e.g. Costa Rica, Honduras,...)
    • Countries that played the previous final of the Central American Cup, get extra berths for the competition
    • Canadian Championship uses correct qualifier teams from League1 Canada (when using my Canada file)
  • No coefficients are used, because unfortunately this seems not to work in FM26 for North America
  • WARNING: Since this file replaces all the in-game CONCACAF competitions, there is a chance that it may not work well together with other editor files or other in-game competitions (though I must say, the in-game version is a poor implementation). Please be aware of the risk. I have designed the rules so they work with my rule files for each country and I recommend to use that setup. If you wish to use this file together with files from other editors, please do some tests before starting the save to observe the behaviour.
  • NOTE: there is a helper competition called "CONCACAF Continental Competitions". This is not a real competition, but only contains stages to setup the qualified teams for Central American and Caribbean Cups. (I left it visible so that I could analyze it in case of issues).
